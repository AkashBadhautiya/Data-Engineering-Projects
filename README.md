# Data Engineering Project (24AI60R43)

This repository contains assignment work for a data engineering course (student ID: 24AI60R43). It includes Jupyter notebooks for various assignments, a Python script demonstrating heap operations, and supporting data files.

## 📁 Repository Structure

### 📂 `24AI60R43_assignment2/`

- `24AI60R43_Assignment2.ipynb` — Colab notebook for Assignment 2 (heap operations and analysis).
- `24ai60r43_assignment2.py` — Python script that reads `data-2col.txt`, builds heaps, finds k-th largest elements, and performs heap sort.
- `data-2col.txt` — Sample two-column dataset used by the script.
- `Assignment1.docx` — Document file (likely related to Assignment 1).

### 🧪 Other Notebooks (per assignment / lab)

- `24AI60R43.ipynb` — Advanced Python data structures and NumPy exercises (lists, tuples, dictionaries, matrix operations).
- `24AI60R43_Assignment3.ipynb` — Data cleaning and exploration using pandas; builds a segment tree to support range queries (sum/max/min).
- `24AI60R43_Assignment5.ipynb` — Loads a timestamped CSV, fills missing values, sorts, and computes statistics (uses pandas + NumPy).
- `24AI60R43_Assignment_08.ipynb` — NLP assignment using spaCy: extracts named entities and POS tags, computes similarity scores between sentences.
- `24AI60R43_Assignment_09 (1).ipynb` — Neo4j graph demo: connects to Neo4j, creates nodes/relationships, and includes example graph queries.
- `24AI60R43_Assignment_3.ipynb` — Zip file extraction + basic file processing; computes word probabilities with Laplace smoothing.
- `24AI60R43_Assignment_5.ipynb` — Builds and analyzes a task dependency graph using an adjacency matrix (JSON input).
- `24AI60R43_Assignment_7_DE_LAB.ipynb` — Spam detection preprocessing: extracts SMS spam collection data, builds a DataFrame, and uses NLTK for NLP tasks.
