This repository contains assignment work for a data engineering course (student ID: 24AI60R43). It includes Jupyter notebooks for various assignments and a Python script demonstrating heap operations on a sample data file.

- `24AI60R43.ipynb`, `24AI60R43_Assignment3.ipynb`, `24AI60R43_Assignment5.ipynb`, `24AI60R43_Assignment_08.ipynb`, `24AI60R43_Assignment_09 (1).ipynb`, `24AI60R43_Assignment_3.ipynb`, `24AI60R43_Assignment_5.ipynb`, `24AI60R43_Assignment_7_DE_LAB.ipynb`

## ✅ How to Run

To run the example heap script:

```bash
python "24AI60R43_assignment2/24ai60r43_assignment2.py"
```

Ensure you run it from the repository root so the script can find `data-2col.txt`.

## 📌 Notes

- The Python script assumes the input file is tab-separated with two columns (key and integer value).
- The script includes functions for building max-heaps, extracting the maximum, finding the k-th largest element, and heap sort.
